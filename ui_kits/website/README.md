# SkillLeo Website UI Kit

Premium dark-mode SaaS website for SkillLeo — agency platform.

## Pages (click-through in index.html)
- **Home** — Hero, features bento grid, how it works, pricing, team, footer
- **About** — Company story, CEO, team stats, registration badges
- **Services** — 10 service cards with hover glow
- **Contact** — Split layout: info + form

## Components
- `SharedComponents.jsx` — Navbar, Footer, Badge, Button, SectionHeader, GlowCard
- `HomePage.jsx` — All home page sections
- `AboutPage.jsx` — About page
- `ServicesPage.jsx` — Services grid
- `ContactPage.jsx` — Contact form + info

## Tech
- React 18 + Babel (inline JSX)
- Lucide icons via CDN
- Geist font via Google Fonts
- CSS variables from `../../colors_and_type.css`
